Online Web Based Examination System

July, 2014 | Lucknow

During my summer vacations I completed a training on ASP.NET with C# & MS SQL Server in Kanpur at the CMC Academy. The project I chose was an Online Web Based Examination System. I designed and developed the web application to facilitate a time bound exam which can be created by anybody for a selected number of applicants.

It supports question database management, multiple choice questions, ranked results and much more.

Featured below are some of the screenshots of the web app.







Pages with context

These pages are basically those with a common content both for the admin and the student.




Admin Section

Once the admin logs in he / she is presented with a dashboard which provides all the relevant instructions on how to create the exam.




Student / Candidate Section

The student then registers and logs in. He / she adds the exam recommended to him by the admin then proceeds to attend the exam. Score obtained can be viewed instantly after the completion of the exam.




Results in Admin Section

The admin can then see the ranked results of all the candidates who have taken the exam after the alloted date for the exam has expired.
